# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.2.0] - 2025-11-28

### Added

- Semantic highlighting support for better type-aware syntax coloring
- Bracket pair colorization with 6 distinct colors
- Bracket pair guides for improved code navigation
- Git decoration colors for modified, added, deleted, renamed, and untracked files
- Complete minimap styling with selection, error, warning, and find match highlights
- Breadcrumb navigation styling
- Search/find match highlighting with distinct colors
- Scrollbar styling with improved visibility
- Editor inlay hints styling for TypeScript/JavaScript parameter hints
- Ghost text styling for Copilot suggestions
- Sticky scroll header styling
- Debug icon colors for breakpoints and controls
- Testing icon colors for test results
- Symbol icon colors for better outline view
- Diff editor colors for comparing files
- Merge conflict colors for resolving conflicts
- Peek view styling
- Notification colors
- Quick picker styling
- Command center styling
- Settings editor styling
- Welcome page styling
- Menu and menubar colors
- Charts colors
- Extension badge colors

### Changed

- Improved editor gutter colors for better git diff visibility
- Enhanced status bar with debugging and error/warning indicators
- Better terminal styling with selection background
- Improved panel styling with active border indicator
- Enhanced input validation colors
- Better button colors with hover states

### Fixed

- Added missing tab active border top indicator
- Added missing editor group header background
- Added missing tree indent guides
